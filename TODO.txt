Roadmap

1. Реализовать меню
    - Создать свою конфигурацию template.yml
    - Развернуть машины исходя из своей конфигурации template.yml
    - показать список машин
    - Удалить развернутые машины по списку пользователей.
2. В template.yml должно быть учтено следующее:
    - имя шаблона
    - name имя машины
    - vmid клонируемой машины
    - full 0/1 связанное или полное клонирование.
    - параметры CPU, memmory, HDD, serial и тд берутся из шаблона
    - сетевые параметры указываются по принципу к какому бриджу подключены (пример: hq-cli)
3. Разворачивание машин:
    - список пользователей должен брать из файла users.txt. Формат записи user@pve
    - на каждое поле из списка users.txt создается пользователь/пароль и отдельный пул. Назначаются права на пул.
    - проверка на наличие пользователя/пула
    - имена машин должны браться из teplate.yml -> name
    - vmid береться последовательно начиная с 1000, доступна проверка на "занятость"
    - машины создаются равномерно на всех нодах кластера / на текущей ноде
    - создать машины клоном, full=0/1 брать из template.yml
4. Редактирование машин
    - настройка сетевых адаптеров
5. Удаление машин
    - удалять машины по списку пользователей / удаление пула и всех связанных машин / удаление пользователя
